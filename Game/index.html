Блудный рыцарь

Созданный группой ПОСс-13Б

Елена Митько, Александр Фильчак, Роман Туаев, Максим Лебедев,

Обзор

  • Цель - Вывести рыцаря из лабиринта. Чтобы выйти из лабиринта, необходимо найти обычную дверь. Чтобы открыть магическую дверь, надо найти 3 энергии для ее открытия. Чтобы выйти из мира духов, надо найти портал. Собирайте бонусы, чтобы получить очки.

  • Управление - Управлять персонажем можно с клавиатуры, так и с помощью мышки. По умолчанию для передвижения с помощью клавиатуры используются кнопки «left», «up», «right», «down». Для передвижения с помощью мышки используются на игровом панели кнопки «left», «up», «right», «down».

  • Сохранение - Данные игры сохраняются автоматически после прохождения уровня.

  • Счет - Чем выше, тем лучше. Проходить уровни быстрее, чтобы получить более высокий балл.
