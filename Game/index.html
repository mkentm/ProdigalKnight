Обзор

  • Цель - Вывести рыцаря из лабиринта. Чтобы выйти из лабиринта, необходимо найти дверь и ключи для ее открытия. Время для поисков ограничены. Собирайте бонусы, чтобы получить преимущества: Часы
    Добавляется определенное количество времени к таймеру    , Факел
    Освещает часть лабиринта    , и Монета
    Монеты для приобретения обновления    .
  • Управление - чтобы двигаться. чтобы поставить/забрать факел. чтобы приостановить.
  • Сохранение - Данные игры сохраняются автоматически в конце уровня обновления и после покупки.
  • Счет - Чем выше, тем лучше. Проходить уровни быстрее, и редко использовать факелы, чтобы получить более высокий балл.
